## 补充Prompt：视觉规范与技能页设计

### 一、字体层级系统修订 (Typography Revision)

```
字号规范（基于16:9页面，1920×1080参考）：
- 页面大标题：思源黑体 Bold / Montserrat Bold，字号 48-56pt，字重 700，行高 1.2
- 项目副标题：思源黑体 Medium / Inter Medium，字号 24-28pt，字重 500，透明度 90%，行高 1.4
- 核心内容文案：思源黑体 Regular / Inter Regular，字号 18-20pt，行高 1.6
- 标签/注释：思源黑体 Regular / Inter Regular，字号 14-16pt，透明度 70%，行高 1.5

层级控制原则：
- 最大字号与最小字号差距控制在 3-4 级以内（如 56pt → 20pt → 16pt）
- 相邻层级字号差 ≤ 8pt，避免跳跃感
- 统一使用 Regular 字重作为正文基准，避免过细字体（Light/Thin）
```

### 二、配色方案统一修订 (Color Palette - Light Mode)

```
全局浅色模式配色：
- 背景色 (Background)：#F8F9FA 米白微灰，全页面统一
- 卡片背景 (Card)：#FFFFFF 纯白，配 1px 浅灰描边 #E5E7EB，圆角 16px
- 主色 (Primary)：#2563EB 科技蓝 —— 用于标题强调、关键数据、CTA按钮
- 辅色 (Secondary)：#059669 翡翠绿 —— 用于正向数据、成功状态、增长指标
- 强调色 (Accent)：#7C3AED 紫罗兰 —— 用于标签高亮、图标点缀、Hover状态
- 文字色 (Text Primary)：#111827 深炭灰，90%透明度
- 次要文字 (Text Secondary)：#6B7280 中灰，70%透明度

配色应用规则：
- 每页仅使用 Primary + Secondary + Accent 三色组合，禁止额外颜色
- 数据卡片使用白底 + 左侧 4px 色条（蓝/绿/紫区分类型）
- 图标统一使用单色线条风格，Hover时切换为 Accent 色
- 所有阴影使用黑色 5% 透明度，Y轴偏移 4px，模糊 12px
```

### 三、标题规范 (Title Format)

```
标题书写规则：
- 禁止使用任何标点符号（：、|、-、——、()、[] 等）
- 使用空格分隔语义单元
- 英文单词首字母大写，中文与英文间保留空格

标题格式示例：
原：Project 01 | Audio Parsing Workflow：让冗余信息无所遁形
改：Project 01  Audio Parsing  信息降噪工作流

原：AI 沉浸式面试模拟器 (Interview Simulator Agent)
改：Project 02  Interview Simulator  AI面试模拟器

原：LBS 城市情绪漫游指南 - [敏捷开发 Vibe Coding 项目]
改：Project 04  City Vibe  情绪漫游指南
```

### 五、全局组件规范 (Global Components)

```
按钮规范：
- 主按钮：Primary 蓝底白字，圆角 8px，内边距 12px 24px，字号 16pt
- 次按钮：白底 Primary 描边，圆角 8px，Hover 时填充 Primary 10% 透明度
```
