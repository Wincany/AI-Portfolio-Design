## 补充Prompt：视觉设计系统与交互展示深化

### 一、全局视觉设计系统 (Visual Design System)

**1. 配色方案 (Color Palette)**
```
主色调体系：
- 主色 (Primary): 深空灰 #1A1A2E 或 科技靛蓝 #2563EB —— 用于标题、关键数据、核心图标
- 辅色 (Secondary): 薄荷绿 #10B981 或 珊瑚橙 #F97316 —— 用于高亮标签、CTA按钮、数据增长指标
- 背景色 (Background): 
  - 深色模式：炭黑 #0F0F23 + 渐变紫蓝晕染
  - 浅色模式：米白 #FAFAFA + 微灰卡片 #FFFFFF
- 功能色 (Functional):
  - 成功/正向：翡翠绿 #059669
  - 警告/待优化：琥珀黄 #D97706
  - 数据可视化：渐变紫粉 #8B5CF6 → #EC4899

配色原则：
- 每页不超过3种主色，保持高对比度但低饱和度冲突
- 数据卡片使用半透明毛玻璃效果 (Glassmorphism)
- 深色背景配霓虹光晕，浅色背景配柔和投影
```

**2. 字体与排版 (Typography)**
```
字体层级：
- 页面大标题：思源黑体 Bold / Montserrat Bold，字号 32-40pt，字重 700
- 项目副标题：思源黑体 Medium / Inter Medium，字号 18-24pt，透明度 80%
- 正文/数据：思源黑体 Regular / Inter Regular，字号 14-16pt，行高 1.6
- 标签/注释：等宽字体 JetBrains Mono，字号 12pt，用于技术术语

排版规则：
- 严格留白：页边距 ≥ 60px，元素间距 ≥ 24px
- 文字区域不超过页面宽度的 60%
- 关键数据使用超大字号 (48-72pt) 独立成块
```

**3. 图形与图标风格 (Iconography & Graphics)**
```
风格定义：
- 线性图标：2px 描边，圆角端点，单色或渐变色填充
- 插图风格：2.5D 等距视角，微渐变，无描边
- 数据可视化：极简柱状图/雷达图，使用品牌渐变色
- Agent 架构图：使用标准流程图符号，节点圆角矩形，连线虚线流动效果

动态元素（如为数字作品集）：
- 微交互：悬停放大 1.02x，点击涟漪效果
- 数据动画：数字滚动增长，图表逐帧绘制
```

### 二、交互设计与功能展示深化 (Interaction & Feature Showcase)

针对四个核心项目，补充以下**交互细节展示要求**：

**项目一：智能播报分析助手 (Audio Parsing Agent)**
```
交互展示要点：
- 【上传流程】拖拽式文件上传区，支持批量音频/视频，显示转码进度条
- 【处理可视化】波形图与文本时间轴联动，点击文本高亮对应音频段落
- 【降噪对比】左右分栏对比视图：左侧原始文本（标红冗余部分），右侧降噪后结构化摘要
- 【导出交互】一键生成思维导图/会议纪要模板，支持飞书/钉钉/Notion API 同步

需展示界面：
1. 文件上传页：虚线框拖拽区 + 格式支持标签（MP3/WAV/MP4）
2. 处理中页：进度环 + 实时转写预览窗口
3. 结果页：三栏布局（原始文本 | 智能摘要 | 关键时间戳导航）
4. 设置页：降噪强度滑块（0-100%）、专业术语词库自定义输入框
```

**项目二：AI 面试模拟器 (Interview Simulator)**
```
交互展示要点：
- 【角色选择】卡片式角色切换（严厉面试官/温和导师/技术专家），带头像与性格标签
- 【对话界面】类微信聊天布局，支持语音输入转文字，AI 回复带"思考中..."打字机效果
- 【实时反馈】侧边悬浮评分面板，维度包括：逻辑性/表达力/技术深度/匹配度，动态雷达图
- 【复盘模式】对话历史时间轴，支持点击跳转特定问题，查看 AI 点评与参考答案对比

需展示界面：
1. 准备页：JD粘贴区 + 简历上传 + 面试难度选择（初级/中级/高级）
2. 面试中页：头像区 + 对话气泡 + 快捷回复按钮（"请详细说明"/"换个角度"）
3. 评分页：五维雷达图 + 具体建议卡片（可展开查看详细点评）
4. 历史页：日历视图回顾过往模拟记录，支持对比多次面试评分曲线
```

**项目三：饮食规划助手 (Diet Planner)**
```
交互展示要点：
- 【模式切换】底部 Tab 切换：日常模式/减脂模式/增肌模式，图标动态变化
- 【LBS 地图】半屏地图 + 半屏列表，支持手势缩放，商户卡片显示卡路里标签
- 【餐盘可视化】圆形餐盘比例图，蛋白质/碳水/脂肪用不同色块，点击展开具体食材
- 【智能替换】长按菜品弹出"换一换"按钮，AI 推荐同等热量替代方案

需展示界面：
1. 首页：今日摄入环形进度条 + 下一餐推荐卡片 + "重新规划"按钮
2. 地图页：POI 标记聚类 + 筛选器（距离/热量/价格）+ 路线规划按钮
3. 餐单详情页：菜品图片 + 营养成分表 + "不喜欢，换一个"交互
4. 个人中心：身体数据录入（身高/体重/目标）+ 饮食偏好标签云（可多选）
```

**项目四：城市情绪漫游指南 (City Vibe)**
```
交互展示要点：
- 【情绪输入】动态文字输入框，placeholder 渐变提示词（"我想..."），输入时背景粒子动画响应
- 【路线生成】加载动画：地图飞入效果 + POI 标记逐个落下 + 路线渐显绘制
- 【漫游卡片】可翻转卡片设计：正面为地图截图+路线，背面为 AI 生成的诗意描述
- 【社交分享】一键生成竖版海报，支持自定义文案与滤镜（复古/清新/赛博朋克）

需展示界面：
1. 首页：大字号情绪输入框 + 热门情绪标签（"孤独感"/"浪漫"/"探索欲"）
2. 生成中页：情绪词云扩散动画 + "正在理解你的心情..."文案
3. 路线页：地图占比 60% + 左侧 POI 列表（带预计停留时间）+ 底部"开始导航"按钮
4. 卡片页：全屏精美漫游卡片 + 保存/分享/再来一次 悬浮按钮组
```
